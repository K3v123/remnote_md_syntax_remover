# remnote_md_syntax_remover
Removes RemNote's `^^^` syntax from exported Markdown files. Creates clean `_cleaned.md` copies safe for NotebookLM podcasts.

**Created for personal use**. Sharing in case others need a no-fuss solution without writing scripts.
## Why use this?
- ✅ **Reduces token count** for AI tools like NotebookLM
- ✅ **Preserves original files** (creates new cleaned copies)
- ✅ **1-click GUI** - no command line needed
- ✅ Works on Windows/macOS/Linux

## Usage
1. Export notes from RemNote as `.md`
2. Run `remnote_cleaner.py`
3. Select file → Get `_cleaned.md` version

> ⚠️ **Never modifies original files** - safe for academic work!

## Requirements
- Python 3.6+
- Tkinter (comes with Python)
